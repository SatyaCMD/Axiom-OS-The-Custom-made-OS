# AxiomOS v1.0 Release Notes

## Overview
AxiomOS is a custom Linux distribution built on Debian 12 (Bookworm), designed for stability, performance, and a professional aesthetic.

## Key Features

### 🎨 Professional Design
- **New Branding**: Custom "AxiomOS" logo and boot splash.
- **Dark Theme**: Pre-configured **Arc Dark** theme for a sleek, modern look.
- **Interactive GUI**: Enabled "Wobbly Windows" and "Magic Lamp" effects for a fluid user experience.
- **Icon Pack**: **Papirus Dark** icons for consistent and flat design.

### 🚀 Core System
- **Base**: Debian 12 "Bookworm" (Stable).
- **Kernel**: Standard Linux AMD64 Kernel.
- **Desktop Environment**: KDE Plasma (Customized).

### 🛠️ Pre-installed Utilities
- **Firefox ESR**: Secure and stable web browsing.
- **Konsole**: Powerful terminal emulator.
- **Dolphin**: Advanced file manager.
- **Neofetch**: System information tool.

## Installation
1. Boot from the ISO.
2. Select "Install AxiomOS" from the boot menu.
3. Follow the on-screen instructions.

## Known Issues
- This is an initial release. Please report any bugs to the development team.
